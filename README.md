# SGO-Rodovias
A SGO-Rodovias administra ativos rodoviários. Cuida da iluminação, praças de pedágio e equipamentos de pista. O trecho analisado abrange 350 km entre a BR-386 (km 345 a 444) e a BR-116 (km 270 a 496), onde ocorrem falhas operacionais. As ocorrências são classificadas em grupos de tarefas, conforme o tipo de sistema afetado.
